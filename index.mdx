kChat is a no-frills, browser-based chatroom application powered by [Supabase](https://supabase.com). It requires no build step, no bundler, and no framework — just open a browser, pick a room, and start chatting. Rooms are persistent, messages are stored in a Supabase database, and your username and theme preferences are saved locally in your browser.

## How kChat Works

<Steps>
  <Step title="Set up Supabase">
    Create a free Supabase project and run the provided SQL schema to create the `rooms`, `messages`, and `profiles` tables.
  </Step>
  <Step title="Configure your credentials">
    Add your Supabase project URL and publishable API key to `chat.js` and `indextablenames.js`.
  </Step>
  <Step title="Serve the files">
    Host the project on any static file server — GitHub Pages, Netlify, Vercel, or a simple local server.
  </Step>
  <Step title="Open the app">
    Visit `index.html` to see all available rooms, pick one, enter a username, and start chatting.
  </Step>
</Steps>
